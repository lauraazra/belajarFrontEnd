untuk function & directives versi 3 & 4 ada perbedaam

Dulu di v3, kita ribet karena harus bolak-balik antara file CSS buat gaya dan file JavaScript buat konfigurasi warna atau font. Kita perlu panggil tiga perintah @tailwind yang berbeda biar kodingannya jalan.

Nah, di v4, semuanya jadi jauh lebih ringkas. Lu nggak perlu lagi file JavaScript tambahan karena Tailwind sekarang sudah satu jiwa sama CSS. Cukup satu baris @import, dan lu bisa bikin warna custom kayak text-wpu langsung pake variabel CSS biasa. Intinya, v4 bikin lu lebih fokus koding di satu tempat tanpa banyak perintilan file yang bikin pusing.
