JSX
- ekstensi sintaks js yang dapat mendeskripsikan ui seperti html
- meski ada cara lain bikin komponen kebanyakan tetap pakai JSX
- JSX mirip html tapi strict dan bisa nampilin info dinamis

3 aturan JSX
- selalu mengembalikan 1 elemen, makannya harus dibungkus 1 elemen aja
- harus selalu tutup tag, bahkan kalo ga ada penutupnya harus ditutup pakai "/>"
- gunakan camelCase untuk atribut, class berubah jadi className

React

Core Concepts
- Component => 
    - pecahan ui, 
    - bisa jadi potongan kode mandiri yang bisa digunakan kembali seperti lego, 
    - jadi untuk ngubah element ui tinggal ubah komponen saja

    Cara buat Component
        - pakai function pada JS
        - diawali huru besar
        - function ngembaliin elemen ui jadi komponen dan ditulis dengan JSX
        - di call sebagai tag html saat dirender

    Komponen Bersarang
        - Aplikasi dengan banyak komponen
        - dalam komponen bisa ada komponen lainnya
        - misal masukin komponen header ke komponen homepage
        - buat function untuk setiap komponen yang dibuat
    
    Menggunakan kembali Komponen
        - componen bisa digunakan berkali kembali
        - gimana biar pakai komponen sama tapi infonya beda
    
    nampilin data pakai props
        - atribut bisa digunakan untuk mengubah elemen
        - contoh src, img, dll
        - kita bisa kirim properti untuk ngubah informasi pakai props

    props refactor pakai destructuring, ternary operator

    Menambah interaksi dengan state
        - halaman bisa berubah karena ada interaksi oleh user
        - di react, data berubah seiring waktu disebut state
        - state bisa ditambah atau dibah ke komponen manapun
        - react pakai state dan event handler untuk interakktivitas
